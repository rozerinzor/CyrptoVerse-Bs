🚀 CryptoVerse

CryptoVerse, HTML, CSS ve Bootstrap kullanılarak geliştirilmiş modern ve responsive bir kripto para web uygulamasıdır. Kripto paralar hakkında bilgi sunar, popüler coinleri gösterir ve kullanıcı etkileşimi için çeşitli bölümler içerir.

---

## 📸 Önizleme

![CryptoVerse Demo](images/crypto.gif)

---

## ✨ Özellikler

- 📱 Responsive tasarım (mobil, tablet, masaüstü uyumlu)
- 💰 Popüler kripto paralar bölümü
- ⭐ Kullanıcı yorumları (testimonial)
- 🔐 Özellikler bölümü (security, analytics, global access)
- 📬 İletişim formu
- 🎨 Modern ve kullanıcı dostu arayüz
- ⚡ Bootstrap grid sistemi ile düzenli layout

---

## 🛠️ Kullanılan Teknolojiler

- HTML5  
- CSS3  
- Bootstrap 5  
- Font Awesome

## 🚀 Kurulum ve Çalıştırma

Projeyi yerel ortamda çalıştırmak için:

1. Repoyu klonla:

git clone https://github.com/your-username/cryptoverse.git


2. Proje klasörüne gir:

cd cryptoverse


3. `index.html` dosyasını tarayıcıda aç.
